# Анализатор страниц

Простое веб-приложение для анализа веб-страниц на основе Flask.

## Использование
- Откройте приложение в браузере по адресу [https://page-analizer-sbja.onrender.com](https://page-analizer-sbja.onrender.com)
- Введите URL страницы для анализа на главной странице и нажмите кнопку "Анализировать".
- После анализа страницы вы увидите информацию о состоянии URL, заголовке страницы, заголовке первого уровня (H1), мета-описании и дате последней проверки.

## Функции
- Анализ страницы по URL.
- Просмотр последних результатов анализа.
- Отображение информации о состоянии URL, заголовке страницы, заголовке первого уровня (H1), мета-описании и дате последней проверки.
